2 Тримай дані під контролем: резервні копії

домашка
Як здійснюється резервне копіювання для віртуальної машини Virtualbox snapshots Ознайомтесь з ними, без них буде складно. https://www.youtube.com/watch?v=Qte4X-rdr2Q https://www.techrepublic.com/article/how-to-use-snapshots-in-virtualbox/ Архіватори та синхронізатори Налаштуйте резервне копіювання найціннішого (наприклад папка university) або ж всієї системи. Для цього: Почитайте про те як робити бекапи певним інструментом тут чи тут (або в інших джерелах): Позначте себе в табличці. Зробіть бекап Зробіть ще одну копію (звичайну копію не інструментом для бекапу) й поставте її в інше місце (флешка, пам’ять телефона, картка пам’яті) Видальте те, що бекапили Відновіть дані з бекапу Коротко задокументуйте найголовніші кроки у текстовому файлі (а також ресурси які ви використали) Якщо ви обрали метод бекапу з допомогою GUI (графічного додатку), то повинні спробувати хоча б один CLI (термінальний) метод.
